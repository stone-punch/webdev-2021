IntelliJ IDEA 단축키 15가지

  • Alt + Enter
    코드에서 오류가 발견된 경우에 해당하는 오류 위에 커서를 놓고 Alt + Enter를 누르면
    해당 오류에 대한 수정 제안 목록이 표시된다.
  • F2
    F2를 사용하면 수정이 필요한 위치로 쉽게 이동할 수 있다.
  • Alt + 1
    만약 전체화면으로 에디터를 사용하다가 프로젝트 창을 열어서 작업을 해야한다면 Alt + 1를
    통해 편리하게 프로젝트 창을 열고 선택할 수 있다.
  • Esc
    프로젝트 창에서 Esc를 누르면 다시 에디터로 돌아와 코드 작업을 이어갈수있다.
    Esc는 병경사항을 적용하지 않고 모든 팝업을 닫을 때 사용할수도 있다.
  • Ctrl + E
    Ctrl + E를 누르면 최근에 열었던 파일을 확인할 수 있다.
    Ctrl + E를 누르고 방향키를 통해 최근의 파일들을 탐색할 수 있고 키워드 입력으로
    특정항목을 검색할 수 있다.
  • Ctrl + B
    Ctrl + B를 사용하면 심볼의 선언으로 이동할 수 있다.
    필드에서 Ctrl + B를 누르면 필드 선언으로 커서가 이동한다.
    클래스 이름에서 Ctrl + B를 누르면 클래스 파일로 이동한다.
  • Alt + F7
    사용위치를 검색하고자할 때, Alt + F7을 누르면 모든 사용위치가 표시된다.
    인터페이스에서 Alt + F7을 누르면 필드 선언 또는 인터페이스를 구현한 클래스를 비롯한
    해당 인터페이스가 사용된 모든위치가 검색창에 표시된다.
  • Ctrl X 2
    Ctrl을 두번 누르면 Run Anything 창이 열린다.
    Run Anything 창에서 검색을 통해 실행할 항목의 이름을 입력할 수 있다.
  • Ctrl + W, Ctrl + Shift + W
    커서 주변의 코드 섹션을 확장하거나 축소할 수 있다.
    선택 영역 확장을 사용하면 확장된 섹션의 유효한 표현식을 자동으로 선택한다.
    Alt + 아래방향키나 Ctrl + Shift + W를 누르면 유효한 Java블록을 통해 선택영역이 축소되어
    커서가 돌아온다.
  • Ctrl + /
    Ctrl + /을 누르면 코드가 줄 주석으로 처리된다.
  • Shift + Ctrl + Enter
    Shift + Ctrl + Enter를 사용하면 IntelliJ IDEA에서 코드의 구문을 자동완성하고 커서를 적절한
    위치로 이동시킨다.
  • Ctrl + Alt + L
    Ctrl + Alt + L을 사용해서 간편하게 현재 파일의 서식을 프로젝트 표준에 맞출 수 있다.
  • Shift + Ctrl + Alt + T
    Shift + Ctrl + Alt + T를 사용해서 모든 리팩토링을 표시할 수 있다.
    심볼이나 선택영역에서 Shift + Ctrl + Alt + T를 누르면 적용가능한 리팩토링 옵션이 표시된다.
  • Shift + Ctrl + A
    Shift + Ctrl + A를 누르면 IntelliJ IDEA의 모든 액션을 검색할 수 있다.
  • Shift X 2
    Shift를 두번 누르면 무엇이든 검색 가능한 검색창이 표시된다.
